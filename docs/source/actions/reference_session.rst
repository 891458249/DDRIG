=====================================
Reference Session |reference_session|
=====================================

Reference Session Action references another ddrig file into the current ddrig session. The contents of a referenced session cannot be edited.

Example use cases:

- Separating the longer ddrig builds into smaller chunks.
- Combining different parts of the same rig into one. For example putting together Face and Body ddrig files into a full character rig.
